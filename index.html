Types of selectors

#id - #color1 - Selects the element with id="color1" and changes the background color. Selector Specificity: 100.

.class - .color2 - Selects all elements with class="color2" and changes the background color. Selector Specificity: 10.

element.class - p.color3 - Selects only "p" elements with class="color3" and changes the background color. Selector Specificity: 11.

element - p - Selects all "p" elements and changes the background color. Selector Specificity: 1.

element[attribute] - p[data-type] - selects all "p" elements with attribute[data-type] and changes the background color. Selector Specificity: 11.

This "p" element has two clases. Not only the color is changed but also the border too. Selector Specificity: 10 each.

element element - fieldset p - Selects all "p" elements inside "fieldset" elements and changes the background color. Selector Specificity: 2.

This is a paragraph too but it is not a child of the fieldset, it is inside a div.

This is paragraph too.

element > element - fieldset > p - selects all "p" elements that are children of a "fieldset" element and changes the background color. Selector Specificity: 2. It works because in ccs rules it is declared lower.

This is paragraph too.

This is paragraph too.

element+element fieldset + p Selects the first "p" element that are placed immediately after "fieldset" elements and changes the background color. Selector Specificity: 2.

element,element,.. - h2,h3 - Selects all "h2","h3" elements and changes the background color. Selector Specificity: 1.

element1~element2 p ~ ul Selects every "ul" element that are preceded by a "p" element and changes the background color. Selector Specificity: 2.
